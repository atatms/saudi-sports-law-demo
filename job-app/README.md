# وظيفة — تطبيق التوظيف الذكي (iOS & Android)

تطبيق جوال متكامل للبحث عن الوظائف مبني على **React Native + Expo + TypeScript**،
يعمل بنفس الكود على أجهزة **iOS (آبل)** و**Android** والويب. واجهة عربية بالكامل
بتخطيط من اليمين إلى اليسار (RTL).

التطبيق يعيد بناء جميع الشاشات والخيارات الموجودة في ملف التصميم المرفق، مع إضافة
**قائمة منسدلة للمناطق الإدارية في السعودية** لتصفية الوظائف حسب موقعها.

---

## ✨ المزايا (مطابقة للملف + الإضافة المطلوبة)

| # | الشاشة | الوصف |
|---|--------|-------|
| 1 | **الرئيسية** | تحية، بانر اكتشافات الذكاء الاصطناعي، نقاط السيرة الذاتية و ATS، إحصائيات الطلبات والمقابلات، أفضل التطابقات |
| 2 | **اكتشف الوظائف** | بحث + فلاتر (هجين/عن بعد/في الموقع/الراتب) + **قائمة منسدلة للمناطق** + ترتيب حسب تطابق الذكاء الاصطناعي |
| 3 | **تفاصيل الوظيفة** | نسبة التطابق، تطابق المتطلبات، الوصف، المتطلبات، نبذة عن الشركة، شريط تقديم ثابت |
| 4 | **طلباتي** | تبويبات (الكل/النشطة/المقابلات/العروض) + مسار حالة الطلب + إجراءات (قبول/تفاوض/الاستعداد بالذكاء الاصطناعي) |
| 5 | **التعلم** | دورات مقترحة، مهارات تحتاج تطويرها، مؤشر التقدم والإحصائيات |
| 6 | **ملفي** | بيانات الملف، المهارات، الإعدادات (إشعارات/خصوصية/اشتراك/لغة/دعم)، تسجيل الخروج |
| 7 | **محلل السيرة الذاتية** | النتيجة الكلية، تفصيل الأقسام، اقتراحات الذكاء الاصطناعي، إعادة التحليل |
| 8 | **توافق ATS** | النتيجة حسب منصات ATS، الفحوصات التقنية، الكلمات المفتاحية الناقصة |

### 🗺️ القائمة المنسدلة للمناطق (المطلوبة)
تشمل المناطق الإدارية الـ13 في المملكة العربية السعودية مع مدنها الرئيسية:
الرياض، مكة المكرمة، المدينة المنورة، القصيم، الشرقية، عسير، تبوك، حائل،
الحدود الشمالية، جازان، نجران، الباحة، الجوف — بالإضافة إلى خيار "كل المناطق".
المصدر: [`src/data/regions.ts`](src/data/regions.ts).

## 🚀 التشغيل

```bash
cd job-app
npm install
npm start          # ثم امسح رمز QR بتطبيق Expo Go
# أو مباشرة:
npm run ios        # محاكي iOS (يتطلب macOS + Xcode)
npm run android    # محاكي/جهاز Android
npm run web        # متصفح
```

### بناء حزم المتجر (اختياري)
```bash
npm install -g eas-cli
eas build -p ios        # ملف IPA لمتجر آبل
eas build -p android    # ملف AAB لمتجر جوجل
```

### فحص الجودة
```bash
npm run typecheck   # TypeScript بدون أخطاء
```
تم التحقق من أن الحزمة تُبنى بنجاح عبر Metro لكل من iOS و Android.

## 🧱 البنية

```
job-app/
├── App.tsx                  # نقطة الدخول + تفعيل RTL + الحاوية الملاحية
├── app.json                 # إعداد Expo (أيقونات، معرفات، iOS/Android)
└── src/
    ├── theme/               # الألوان والمسافات والظلال
    ├── data/                # بيانات تجريبية (مناطق، وظائف، طلبات، دورات، ملف، ATS)
    ├── utils/               # تنسيق الراتب/نمط العمل
    ├── components/          # مكونات قابلة لإعادة الاستخدام (JobCard، ScoreRing، RegionDropdown ...)
    ├── navigation/          # التبويبات السفلية + المكدّس (Stack)
    └── screens/             # الشاشات الثماني
```

## التقنيات
- **Expo SDK 52** / React Native 0.76 / React 18
- **React Navigation** (تبويبات سفلية + مكدّس)
- **react-native-svg** (الحلقات الدائرية للنتائج)
- **@expo/vector-icons** (Ionicons)
- **TypeScript** (وضع صارم)

> البيانات حالياً تجريبية (mock) داخل `src/data`؛ يسهل استبدالها بطبقة API حقيقية لاحقاً.
