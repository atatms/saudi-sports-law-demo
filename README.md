# منصة متابعة الأبحاث والمجلات العلمية

منصة شخصية تعمل في المتصفح (وعلى الجوال وسطح المكتب) لمتابعة الأبحاث من
الكتابة إلى النشر، وإدارة قاعدة بيانات المجلات القانونية مع ترشيح ذكي
لأنسب المجلة لمقالك.

> صفحة المؤتمر السابقة محفوظة في `conference.html`.

## المزايا

- **تسجيل دخول** باسم مستخدم ورقم سري (يُخزَّن مُجزّأً SHA-256 محليًا)، مع
  ربط البريد الأكاديمي والخاص في الملف الشخصي.
- **لوحة متابعة** فيها حالة يومية وأربعة أعمدة: تحت الكتابة، جاهزة للتسليم،
  مُرسلة، ومجلات تمت مراسلتها.
- **مسار حالات معتمد**: تحت الكتابة ← جاهزة للتسليم ← مُرسلة ← تحت المراجعة ←
  تعديلات مطلوبة ← مقبول ← منشور (أو مرفوض)، مع خط زمني لكل بحث.
- **قاعدة بيانات المجلات** مع تصنيف Web of Science وScopus والربع (Quartile)
  ومدة النشر وحد الكلمات ونمط التوثيق والنطاق.
- **رفع CSV/Excel** لإضافة المجلات دفعةً واحدة (مطابقة الأعمدة تلقائيًا).
- **قارئ الملخص بالذكاء الاصطناعي**: يرشّح أنسب المجلات لملخص مقالك.
  - مع مفتاح **Claude API** (في الإعدادات): تحليل بنموذج `claude-opus-4-8`.
  - بدون مفتاح: تحليل محلي ذكي (تطابق كلمات + ملاءمة حد الكلمات والتصنيف).
- **تنبيهات مستمرة**: عند تسجيل ملاحظة/رد من مجلة يظهر تنبيه ويبقى حتى تردّ
  عليه وتغلقه.
- **ألوان مريحة للعين** + وضع ليلي، **خطوط عربية واضحة** (Tajawal/Cairo)،
  وأسماء المجلات بخط لاتيني، وتصميم متجاوب يعمل على Samsung S24 Ultra
  وسطح المكتب.
- **نسخ احتياطي**: تصدير/استيراد كل البيانات كملف JSON لنقلها بين الأجهزة.

## التشغيل

افتح `index.html` في المتصفح (أو انشره عبر GitHub Pages). كل البيانات
تُحفظ محليًا على الجهاز عبر `localStorage`.

## البنية

| الملف | الوظيفة |
|------|---------|
| `index.html` | هيكل الواجهة |
| `css/app.css` | التنسيقات والألوان والتجاوب |
| `js/store.js` | طبقة البيانات والتخزين المحلي (قابلة للترقية) |
| `js/auth.js` | المصادقة وتجزئة الرقم السري |
| `js/matcher.js` | محرك الترشيح (Claude API + تحليل محلي) |
| `js/importer.js` | استيراد CSV/Excel |
| `js/app.js` | تشغيل الواجهة وربط كل شيء |

## التطوير لاحقًا

المنصة قابلة للتحديث: عند طلب ميزة أو ملاحظة تُعدَّل في نسخة محدثة. يدعم
`store.js` ترقية المخطط عبر `schemaVersion` للحفاظ على البيانات الحالية.
